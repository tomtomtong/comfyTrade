# Product Overview

MT5 Trading Strategy Executor is an Electron desktop application that provides a comprehensive trading platform for MetaTrader 5. The application enables users to build visual trading strategies using a node-based editor, import historical data for backtesting, and execute trades with advanced risk management.

## Core Value Proposition

- **Visual Strategy Building**: Node-based drag-and-drop interface for creating trading strategies without coding
- **MT5 Integration**: Direct connection to MetaTrader 5 for real-time trading and data access
- **Historical Backtesting**: Import and test strategies against historical data from MT5 or CSV files
- **Risk Management**: Built-in pip-based loss calculations and position management tools
- **Real-time Trading**: Execute trades with customizable parameters and monitor positions

## Key Features

- Node editor with triggers, indicators (MA, RSI), conditional logic, and trade execution
- Historical data import from MT5 (8 timeframes) or CSV files with persistence
- Symbol input with autocomplete and MT5 symbol fetching
- Real-time account monitoring and position management
- Trailing stop functionality and risk warnings
- Backtest mode with visual indicators

## Target Users

Forex and financial market traders who want to:
- Build and test trading strategies visually
- Automate trading execution with MetaTrader 5
- Backtest strategies against historical data
- Manage risk with proper position sizing
